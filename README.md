# AI Research Paper Intelligence Platform (Roo Code Workflow)

Homework 1 – Let's Coding in AI

This repository demonstrates how an AI coding agent (Roo Code) can assist in developing a software system using a structured workflow.

The goal of this project is to build a simple **AI Research Paper Intelligence Platform** that allows users to search for recent AI-related research papers and display them in a web interface.

---

# Project Goal

The objective of this assignment is to explore how AI coding agents can assist software development through multiple steps instead of generating everything in a single prompt.

The development workflow includes:

1. Writing a system design document
2. Generating a system architecture
3. Implementing the system using AI orchestration
4. Debugging issues using AI prompts

---

# Development Workflow

## Step 1 – System Design

A design document (`DESIGN.md`) was written to describe:

- system goals
- major components
- user interaction
- backend functionality

This document serves as the foundation of the system.

---

## Step 2 – Architecture Generation

Using **Roo Code Architect mode**, the system design was converted into a detailed architecture specification (`ARCHITECT.md`).

The architecture includes:

- frontend modules
- backend services
- API structure
- system data flow

---

## Step 3 – System Implementation

Using **Roo Code Orchestrator mode**, the AI agent generated the application according to the architecture specification.

The system includes:

- backend API
- frontend interface
- research paper search functionality

---

## Step 4 – Debugging

Some issues appeared in the generated code. Additional prompts were used to fix errors without modifying the system design.
